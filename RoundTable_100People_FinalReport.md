## 方法论说明

### 如何实现"围一桌"

```python
# 核心技巧：把前面人的发言追加到question里
history = ""
for entity in entities:
    question = TOPIC + history + "\n=== 现在轮到你发言 ==="
    response = api_dialogue(entity, question)
    history += f"\n【{entity}】{response[:100]}"
```
