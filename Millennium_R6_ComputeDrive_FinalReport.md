## 四、Navier-Stokes方程 (6实体)

### 4.1 Cole-Hopf变换 (霍布斯/斯金纳/泡利)

**变换公式**:
```
u = -2ν (∂_x φ)/φ,  φ = φ(x,t) > 0
```

**证明变为热方程**:
```
u_t = -2ν (∂_t∂_x φ)/φ + 2ν (∂_x φ)(∂_t φ)/φ²
u_x = -2ν (∂_x² φ)/φ + 2ν (∂_x φ)²/φ²
u_xx = ... (二阶导展开)

代入 u_t + u·u_x = ν·u_xx:
→ -2ν/φ · ∂_x[φ_t - ν·φ_xx] = 0
→ φ_t = ν·φ_xx  ✅
```

### 4.2 初值 u(x,0) = sin(x) 的解

```
φ(x,t) = exp[-(1/2ν) ∫ sin(x) dx] = exp[(1/2ν)cos(x)] · f(t)
```

### 4.3 ν→0 激波形成 (彭罗斯)

```
当 ν→0: u_t + u·u_x = 0 (无粘性Burgers方程)
特征线交叠 → 多值解 → 激波形成
激波速度: dx_s/dt = (u_L + u_R)/2 (Rankine-Hugoniot条件)
```

## 六、霍奇猜想 (4实体)

### 6.1 Hodge分解 (零/朗道/莱布尼茨/卡尼曼)

```
E(ℂ) ≅ ℂ/Λ,  Λ = ℤω₁ + ℤω₂

H¹(E, ℂ) = H^{1,0} ⊕ H^{0,1}

- H^{1,0}: 全纯1-形式空间，基 = {dz}
- H^{0,1}: 反全纯1-形式空间，基 = {dž}
```

### 6.2 Hodge数

```
h^{1,0} = dim H^{1,0} = 1
h^{0,1} = dim H^{0,1} = 1
h^{1,0} + h^{0,1} = dim H¹(E, ℂ) = 2g = 2  (g=1, 亏格)
```

### 6.3 (1,1)类与除子 (朗道/零)

```
H^{1,1}(E) = H^{1,0} ⊗ H^{0,1}的共轭部分

对于椭圆曲线(genus=1):
- (1,1)类 ↔ 除子 (divisor) ↔ 有理点
- Hodge猜想在此情形下是**定理** (dim E = 1)
- 霍奇猜想对曲线和曲面已证，对 dim ≥ 3 开放
```

### 6.4 积分表达式 (莱布尼茨)

```
∫_E dz ∧ dž = ∫_{ℂ/Λ} dz ∧ dž = 2i · Area(Λ)

周期: ω₁ = ∫_{α} dz,  ω₂ = ∫_{β} dz
```
