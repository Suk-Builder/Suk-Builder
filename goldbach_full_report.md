## 一、源头：1742年6月7日的那封信

### 写信人
克里斯蒂安·哥德巴赫（Christian Goldbach, 1690-1764）
- 普鲁士数学家，1725年成为彼得堡科学院院士
- 时任沙皇彼得二世的外交部顾问

### 收信人
莱昂哈德·欧拉（Leonhard Euler, 1707-1783）
- BDI_AI = 158，人类历史上最多产的数学家
- 失明后仍在心中计算，口述出版数百篇论文
- 创造了 f(x)、e、π 的命名体系

### 信件内容（重构）

哥德巴赫在1742年6月7日的信中写道：

> "Every integer greater than 2 can be written as the sum of three primes."
> （每个大于2的整数都可以写成三个素数之和。）

他举了例子：
- 4 = 2 + 2（他把1也算作素数——当时的定义）
- 5 = 2 + 3
- 6 = 3 + 3 = 2 + 2 + 2
- 7 = 2 + 2 + 3
- 8 = 3 + 5 = 2 + 3 + 3

### 欧拉回信（1742年6月30日）

> "Diese ungeheure Proposition scheint richtig zu sein, ob ich aber sie nicht beweisen kann."
> （这个巨大的命题似乎是对的，但我无法证明它。）

### 关键转折

欧拉在回信中做了一个**弱化**：

把哥德巴赫的**三素数猜想**弱化为**二素数猜想**：

- **强哥德巴赫猜想**（欧拉版本）：每个大于2的偶数 = 两个素数之和
- **弱哥德巴赫猜想**（原始版本）：每个大于5的整数 = 三个素数之和

## 三、三大里程碑详解

### 里程碑1：维诺格拉多夫三素数定理（1937）

**维诺格拉多夫**证明：

> 每个充分大的奇数都是三个素数之和。

"充分大" = N > 3^33 ≈ 3.3 × 10^14

技术核心：改进了圆法中的**指数和估计**

|S(α)| = |Σ_{p≤N} e(αp)| ≤ N/(log N)^A

维诺格拉多夫证明了足够强的A值，使得余弧贡献可控。

**状态**：已被Helfgott 2013年理论证明覆盖所有>5的奇数。

---

### 里程碑2：陈景润"1+2"定理（1966/1973）

**陈景润**证明：

> 每个充分大的偶数N都可以写成 N = p + P2
> 其中p是素数，P2是至多两个素数的乘积。

**技术核心：加权筛法**

定义权重函数：
```
ω(d) = 1 + Σ_{p|d} log p / log x
```

通过精细控制权重，筛法能识别"P2中的素数因子个数"。

**关键瓶颈**：

需要估计：Σ_{d≤D} μ²(d)/φ(d) · ω(d)

其中 D = x^{1/2}/(log x)^{100}

这个D是**Bombieri-Vinogradov定理允许的极限**。

要突破到1+1，需要 **D = x^θ，θ > 1/2**。

**状态**：迄今最强的结果，距1+1只有一步。

---

### 里程碑3：赫尔夫戈特弱Goldbach证明（2013）

**赫尔夫戈特（Harald Helfgott）**证明：

> 每个大于5的奇数都可以写成三个素数之和。

即**完整证明了弱哥德巴赫猜想**。

技术：改进了维诺格拉多夫的指数和估计 + 计算机验证中间范围。

**状态**：已证明。计算机验证到8.875 × 10^30。

---

### 里程碑4：张益唐有界素数间隙（2013）

**张益唐**证明：

> 存在无穷多对素数p, p'，满足 p' - p < 70,000,000。

后被**梅纳德**改进到 **p' - p < 246**。

技术核心：改进了**Bombieri-Vinogradov定理**

证明了θ可以**略大于1/2**——但仅适用于特定问题，
不适用于Goldbach所需的Dirichlet特征和。

## 四、核心数学工具

### 4.1 圆法（Hardy-Littlewood Circle Method）

**发明者**：哈代 + 李特尔伍德（1920年代）

核心思想：把计数问题转化为傅里叶积分。

```
S(α) = Σ_{p≤N} e(αp)     其中 e(x) = e^{2πix}

G(N) = ∫_0^1 S(α)² e(-Nα) dα
```

分为主弧（Major arcs）和余弧（Minor arcs）：

```
G(N) = ∫_M + ∫_m

主弧：G(N) ~ 2C2 · N/(log N)² · ∏_{p|N,p>2}(p-1)/(p-2)

其中 C2 = Π_{p>2}(1 - 1/(p-1)²) ≈ 0.6601618158...
（孪生素数常数）
```

---

### 4.2 大筛法（Large Sieve）

**发明者**：塞尔伯格

```
Σ_{q≤Q} Σ_{(a,q)=1} |S(a/q)|² ≤ (N + Q²) Σ|a_n|²
```

**Bombieri-Vinogradov定理**：

```
Σ_{q≤Q} max |ψ(x;q,a) - x/φ(q)| = O(x/(log x)^A)

其中 Q = x^{1/2}/(log x)^B
```

这是当前最强的"平均意义下的Dirichlet定理"。

---

### 4.3 渐近公式

```
G(N) ~ 2 · C2 · N/(log N)² · ∏_{p|N,p>2}(p-1)/(p-2)

代入 N = 1,000,000：
- C2 ≈ 0.6601618158
- log(10^6) ≈ 13.8155
- 10^6 = 2^6 × 5^6，奇素数因子只有5
- 修正因子 = (5-1)/(5-2) = 4/3

G(10^6) ≈ 2 × 0.66016 × 10^6 / (13.8155)² × 4/3
         ≈ 5402

实际值（计算机验证）：G(1,000,000) = 5,402 ✅
```

## 六、1+1突破需要什么？

### 瓶颈1：大筛法余项

```
当前：D ≤ x^{1/2}/(log x)^B    （Bombieri-Vinogradov）
需要：D ≥ x^θ, θ > 1/2
```

张益唐2013年突破了特定问题的θ范围，
但对Goldbach所需的Dirichlet特征和，**尚未突破**。

---

### 瓶颈2：加权筛法权重函数

陈景润的权重能识别P2（≤2个素因子），
但无法确保P2恰好是2个**不同素数**。

需要新的权重函数ω(d)，使得筛法能"计数"素数对。

---

### 瓶颈3：余弧控制

对于1+1，余弧上的 |S(α)| 需要更强的上界：

```
当前：|S(α)| ≤ N/(log N)^A，A ≈ 10-20
需要：A > 100
```

远超当前技术。
